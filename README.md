# Web Practice Portfolio

**Web_practice** — это рабочая папка для практики фронтенд-разработки. Проект содержит задания по HTML, CSS и JavaScript, распределённые по дням, а также финальный многостраничный проект.

## Структура проекта

- `Day1/` — базовая HTML-верстка с демонстрацией изображения.
- `Day2/` — практика HTML и CSS для простой страницы.
- `Day3/` — верстка и стили с дополнительным `README.md`.
- `Day4/` — две недельные работы: `4-4.4/` и `4-5-4.6/` с графикой и адаптивной страницей.
- `Day5/` — два проекта: `5.2/` и `5.4/`; `5.4/` содержит статические шрифты.
- `Day6/` — страницы для `6.1-6.3/` и `6.5/` с графикой и оформлением.
- `Day7/` — HTML и JavaScript упражнения.
- `Day8/` — HTML и JavaScript с комментариями в `README.md`.
- `Day9/` — HTML и JavaScript практика.
- `Day10/` — интерактивное задание на JavaScript.
- `Day11/` — упражнение `12.1/` с готовой страницей и набором ресурсов.
- `Day13/` — две работы: `13.2/` и `13.3/` с JavaScript-скриптами.
- `Final_Project/` — финальный многостраничный проект новостного сайта.

## О финальном проекте

`Final_Project/` представляет собой законченный фронтенд-сайт с несколькими страницами:

- `index.html` — главная страница сайта.
- `login.html` — страница входа пользователя.
- `register.html` — страница регистрации нового пользователя.
- `news.html` — страница списка новостей.
- `create.html` — форма создания новой записи.
- `edit-news.html` — страница редактирования новости.
- `categories.html` — страница списка категорий.
- `create-category.html` — форма добавления категории.
- `edit-category.html` — страница редактирования категории.

### Стили и логика

- Папка `css/` содержит стили для каждой страницы:
  - `index.css`, `login.css`, `news.css`, `create.css`, `categories.css`, `user.css`, `details.css`, `header.css`, `button.css`
- Папка `script/` содержит JavaScript-файлы для управления поведением сайта:
  - `index.js` — логика главной страницы.
  - `login.js`, `register.js` — обработка форм входа и регистрации.
  - `news.js`, `create.js`, `edit.js` — управление новостями.
  - `categories.js`, `create-category.js`, `edit-category.js` — управление категориями.
- Папка `img/` содержит иконки и графические элементы: логотип, стрелки, иконки для действий.

### Что демонстрирует финальный проект

- Многостраничную структуру сайта.
- Оформление разных состояний интерфейса.
- Взаимодействие с формами создания, редактирования и фильтрации.
- Использование CSS для стилизации страниц.
- Разделение логики на отдельные JavaScript-модули.
